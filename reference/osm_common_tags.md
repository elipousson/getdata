# Common OpenStreetMap tags

A subset of tags scraped from the OpenStreetMap Wiki page on Map
features: <https://wiki.openstreetmap.org/wiki/Map_features>. Only those
tags with a url are included in this reference table.

## Usage

``` r
osm_common_tags
```

## Format

A data frame with 272 rows and 5 variables:

- `key`:

  Key

- `value`:

  Value

- `description`:

  Description of tag/usage

- `category`:

  Category

- `url`:

  OSM Wiki url
